Инструкция по запуску
1) склонировать репозиторий
  git clone https://github.com/BlagochevskyIvan/Game-Of-Life

2) Зайти в папку Game-Of-Life
  cd Game-Of-Life

3) Через Makefile скомпилировать программу
   make all

4) Либо запускаем код с готовыми файлами
   make run1
   make run5
   
   Либо вводим свои стартовые клетки
   Сначала количество, затем координаты x y клетки
